# Address API Spec

## Create Address

Endpoint : POST /api/contacts/{idContact}/addresses

Request Header :
- Authorization : token

Request Body :

```json
{
  "street" : "jalan",
  "city" : "kota",
  "province" : "provinsi",
  "country" : "negara",
  "postal_code" : "234234"
}
```

Response Body :

```json
{
  "data" : {
    "id" : 1,
    "street" : "jalan",
    "city" : "kota",
    "province" : "provinsi",
    "country" : "negara",
    "postal_code" : "234234"
  }
}
```

## Remove Address

Endpoint : DELETE /api/contacts/{idContact}/addresses/{idAddress}
